# Compilational Authorship Manifesto

This repository hosts the **Compilational Authorship Manifesto**, proposing a new perspective on authorship in the age of AI.

The manifesto argues that AI is not an author but a **compiler of human creativity**, and that both AI developers and the authors of training data should be credited.
